# wotchacooking.com-235-Project-2025

In 2025, my teammates Ben, Owen, and I built this recipe web app as a submission for our CS235 course at the University of Auckland. The app uses a CSV of recipes provided by our lecturer and displays them as a browsable collection. Users can search and browse recipes, view full ingredient lists and step-by-step instructions, and create an account to save their favourites.

We had a bigger vision for the project beyond the assignment requirements. Features we planned but didn't get to finish include customisable user profiles, a social following system to see what your friends are cooking, and the ability to share recipes with others. These are things we may come back to and build out properly.

### Tech Stack
- Python / Flask
- SQLAlchemy (ORM)
- HTML/CSS/JS frontend
- CSV data source

### Features
- Browse and search recipes
- View ingredients and step-by-step instructions
- User account creation and login
- Save, comment and view favourite recipes
